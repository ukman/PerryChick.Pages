Privacy Policy

Last updated: April 18, 2025

This Chrome extension, Perry Chick for Youtube, does not collect, store, or transmit any personal data.

What the extension does:

  • Extracts auto-generated subtitles from the currently open YouTube video.
  • Prepares a prompt using the subtitles and opens a ChatGPT tab (if available).
  • No data is transmitted to any external server.
  • Only the user can submit the prompt to ChatGPT — the extension does not send anything automatically.

Data access and usage:

  • The extension reads subtitles from the YouTube page content.
  • It may read from the ChatGPT input area to help pre-fill the message.
  • No data is logged, stored, or sent to third parties.
